Top 5 Video Games of all Time!

Below are listed the top five Video Games of all time by the opinion of Kenny Calhoun (of course):

    1. Fortnite

This game is rated #1 because it contains three games in one. One is Save the World. This gamemode allows you to gain resources used to make weapons and this gamemode allows you to think stratagy. The second gamemode is an online batte royal free-for-all. And the third gamemode is a place where you can create your very own games.

    2. Minecraft

This game is both a survival and creative game which makes it double the fun. It contains special objects which also allows you to create your very own games.

    3. Super Smash Bros: Ultimate

This game is very fun to play with others. THis game features chracters which you eventually become a master of that fight against each other in a brawl. It is a very high graphic game that shows every detail of each character's attacks very clearly.

    4. Lego Batman 3

This game had a storymode that went to many iconic locations. You could play over 50+ characters. There were large planets that you could explore and amazing mini-games.

    5. Star Trek Legends

This game was very adicting and fun. It is an RPG game of stratagy. You fought off against other Star Trek characters. You upgraded your very own characters and progressed through a campaign.
